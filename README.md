# Student Management System
A Student Management System is a Python command-line application used to manage student records efficiently using Object-Oriented Programming and JSON file handling.
## Features
- Add Student Records
- View Student Details
- Update Student Information
- Delete Student Records
- JSON File Storage
- Input Validation
## Technologies Used
- Python
- JSON Module
- Command Line Interface (CLI)
- Object-Oriented Programming (OOP)
## How to Run
1. Install Python
2. Save the file as `student_management.py`
3. Open terminal or command prompt
4. Run the program:
```bash
python student_management.py
